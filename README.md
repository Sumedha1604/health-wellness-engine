A production-quality full-stack AI-powered recommendation system that provides personalized health and wellness recommendations using Machine Learning.

### Frontend
- React
- Vite
- React Router
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js
- JWT Authentication

### Machine Learning
- Python
- Scikit-learn
- Pandas
- NumPy

### Database
- MySQL

### DevOps
- Docker
- Docker Compose
- GitHub Actions
